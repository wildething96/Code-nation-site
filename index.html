Basic Structure of HTML

The basic HTML structure can be quickly auto completed using emmet. Just type ! then hold Shift and hit Enter

The <head> contains all the metadata (document title, character set, styles, links and scripts), specific information about the web page that is not displayed to the user.

The <title> will contain the title of your document which will be displayed in the tab header at the top of the page

The <body> contains all the main content of the website which will be displayed to the browser

The <div> tag is used in HTML to make divisions of content in the web page and to group content together

The <p> tag represents a paragraph of text 

<!doctype html>
    <html>
        <head>
            <meta charset="utf-8">
            <title>test page</title>
        </head>
        <body>
            <div>
                <p>Hello World.</p>
            </div>
        </body>
    </html>

Creating lists and links in HTML

Using a class such as in <div class = "container"> allows to create a group that can all be targeted by the same CSS style properties.

<ul> is one type of list which creates an unordered list, the other type is an ordered list <ol> which will be numbered

The <li> tag is used inside a list and is used for each separate item inside a list

The <a> tag is put around anything that will be linked to another Page whether that be another website or a different pag from your website.

Creating HTML tables

The <table> tag creates a table which by default doesn't have any borders.

A table will have normally have multiple rows inside it written as <tr>

Each row will have cells inside it, the first row of a table will often be a header for each column written as <th>,

Latsly each row will have table data cells written as<td> normally with some information inside them

<body>
  <div class = "container">
    <table>
        <tr>
          <th>Title1</th>
          <th>Title2</th>
          <th>Title3</th>
        </tr>
        <tr>
          <td>item1</td>
          <td>item2</td>
          <td>item3</td>
        </tr>
    </table>
  </div>
</body>
